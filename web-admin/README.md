# web-admin

슈퍼관리자 웹 콘솔 (React + Vite).

## IA
- `/apk`: APK관리 (업로드, 목록, 상세)
- `/devices`: 기기관리 (목록, 검색, 지도, 상세/원격명령)

## 실행
```bash
cd /Users/nahooni0511/workspace/sistrun-hub/web-admin
npm install
npm run dev
```

브라우저에서 기본 `http://localhost:5173` 접속 후 API URL/토큰 입력.
